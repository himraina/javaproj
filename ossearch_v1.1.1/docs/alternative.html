Building OSSearch™ v1.1.1 without Ant:

From the 'ossearch_v1.1.1' directory, compile all java files. 
> cd /ossearch/install/path/ossearch_v1.1.1
> javac -classpath "./src:lib/mail.jar:lib/activation.jar:lib/jsdk.jar:lib/jh.jar" src/com/nebulais/ossearch/*.java
> javac -classpath "./src:lib/mail.jar:lib/activation.jar:lib/jsdk.jar:lib/jh.jar" src/*.java
Note: On Windows replace ':' with ';'

Installing OSSearch™ v1.1.1 without Ant:

  1. Upload (or copy) contents of the 'classes' directory to the servlet directory.
  2. Upload (or copy) contents of the 'template' directory to the servlet directory.
  3. Upload (or copy) the 'data' directory to the servlet directory.
The resulting files and directories on the web server should look as follows: 
/[SERVLET_DIR]/
    OSSearchAdminServlet.class
    OSSearchServlet.class
    ossearch_1.html
	
    com/
        nebulais/
            ossearch/
                *.class
                server/
                    *.class
	
    data/
        ossearch/
            ossearch.cfg
